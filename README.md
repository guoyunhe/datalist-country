# @datalist/country

## Usage

```
npm i -S @datalist/country
```

```js
import { countryDataList } from '@datalist/country';
```

## Example

```json
[
  {
    "name": "Finland",
    "code": "FI",
    "alpha3Code": "FIN",
    "numericCode": "246",
    "topLevelDomains": [".fi"],
    "currencyCodes": []
  }
]
```

## Data Source

1. Country codes: https://en.wikipedia.org/wiki/List_of_ISO_3166_country_codes
2. Currency codes: https://en.wikipedia.org/wiki/ISO_4217
